🌱 基于 STM32 的智能浇花系统

STM32F103C8T6 + ESP8266 + OneNet + 微信小程序

GitHub 仓库

项目概述

本设计基于 STM32F103C8T6 主控芯片，结合土壤湿度 ADC+DMA 数据采集、OLED 显示、自动浇水与手动按键控制、水位检测报警等多种功能，实现了完整的智能浇花系统。

通过 ESP8266 WiFi 模块接入 OneNet 云平台，用户可使用微信小程序远程监控土壤湿度、控制浇水开关，真正实现物联网远程管理。
